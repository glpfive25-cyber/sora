# 默认角色快速选择功能

## ✅ 已添加

在"角色视频生成"页面的提示词输入框下方，添加了三个默认角色的快速选择按钮。

## 🎭 默认角色

### 1. @lahdevqrp.skybounddu
- **图标**: 👤 用户图标
- **颜色**: 紫色渐变
- **说明**: 你之前创建的角色

### 2. @ggvdnqdfn.grizzlehat - 光头强
- **图标**: 👔 领带图标
- **颜色**: 粉红渐变
- **说明**: 光头强角色

### 3. @tbqtyjnvb.bramblebea - 熊大
- **图标**: 🐾 爪印图标
- **颜色**: 蓝色渐变
- **说明**: 熊大角色

## 🎯 使用方法

### 方式 1: 点击快速选择按钮

1. 打开"角色视频生成"页面
2. 在提示词输入框下方看到三个彩色按钮
3. 点击任意角色按钮
4. 角色用户名会自动添加到提示词开头
5. 继续输入动作描述
6. 点击"生成视频"

### 方式 2: 手动输入

直接在提示词中输入：
```
@ggvdnqdfn.grizzlehat 光头强在森林里砍树
```

或

```
@tbqtyjnvb.bramblebea 熊大在树林里奔跑
```

## 🎨 按钮特性

### 视觉效果

- **渐变背景**: 每个角色有独特的颜色
- **悬停效果**: 鼠标悬停时按钮上浮并显示阴影
- **点击效果**: 点击时按钮下沉
- **图标**: 每个角色有专属图标

### 交互反馈

- ✅ 点击后显示成功提示
- ✅ 自动聚焦到输入框
- ✅ 防止重复添加同一角色
- ✅ 智能插入位置（开头）

## 🔧 功能细节

### 智能插入

```javascript
// 如果提示词为空
输入: ""
点击"光头强" → "@ggvdnqdfn.grizzlehat "

// 如果提示词不为空
输入: "在森林里砍树"
点击"光头强" → "@ggvdnqdfn.grizzlehat 在森林里砍树"

// 如果已包含该角色
输入: "@ggvdnqdfn.grizzlehat 跳舞"
点击"光头强" → 提示"提示词中已包含此角色"
```

### 防重复

系统会检查提示词中是否已包含该角色，避免重复添加。

## 🎬 完整工作流程

### 1. 创建角色（如果还没有）

- 上传视频或使用 URL
- 设置时间戳
- 创建角色

### 2. 生成角色视频

1. 切换到"角色视频生成"
2. 点击快速选择按钮（例如"光头强"）
3. 继续输入动作：`在森林里砍树`
4. 完整提示词：`@ggvdnqdfn.grizzlehat 在森林里砍树`
5. 选择参数（模型、时长、方向）
6. 点击"生成视频"
7. 几秒钟后视频出现

### 3. 下载和分享

- 播放视频
- 点击下载按钮
- 或复制视频链接分享
