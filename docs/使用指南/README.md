## 🎯 核心要点

### 视频模型推荐

| 模型 | 推荐度 | 说明 |
|------|--------|------|
| `sora_video2` | ⭐⭐⭐⭐⭐ | **强烈推荐**，成功率最高 |
| `sora_video2-landscape` | ⭐⭐⭐⭐ | 横屏视频 |
| `sora_video2-portrait` | ⭐⭐⭐⭐ | 竖屏视频 |
| `sora_video2-*-15s` | ⭐⭐ | 15秒版本，容易超时 |

### 关键提示

1. **优先使用标准模型** - `sora_video2` 成功率 95%+
2. **避免15秒版本** - 生成时间长，容易504超时
3. **简化提示词** - 10-30字最佳
4. **图生视频避免真人** - 会被审查拒绝
5. **观察实时进度** - 系统会显示生成进度

## 🔍 常见问题

### Q: 504 超时错误怎么办？
**A:** 选择标准模型 `sora_video2`，避免使用15秒版本。详见 [视频生成使用指南](./使用指南/视频生成使用指南.md) 的故障排除章节。

### Q: 如何查看实时进度？
**A:** 前端已自动启用，生成时会显示进度条和百分比。技术细节见 [实时进度功能说明](./技术文档/实时进度功能说明.md)。

### Q: 为什么生成失败？
**A:** 可能原因：
- 提示词违规（暴力、色情、版权）
- 图片涉及真人
- 服务器负载高

详见使用指南的"常见问题解决"章节。

### Q: API 格式是什么？
**A:** 使用标准 Chat API 格式（`/v1/chat/completions`）。详见 [API 重构技术摘要](./技术文档/VIDEO_API_REDESIGN_SUMMARY.md)。
