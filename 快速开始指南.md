# 快速开始指南

## 🚀 三步开始使用

### 1. 启动应用
```bash
npm install
npm start
```
应用将在 http://localhost:3000 启动

### 2. 开始创作
- ✅ **无需配置**：系统已内置免费 API，直接使用即可
- 🎬 **视频生成**：输入描述，选择模型，点击生成
- 🖼️ **图像生成**：切换到图像工具，输入描述即可
- 👤 **角色功能**：切换到角色工具，上传视频创建角色

### 3. 自定义配置（可选）
如果你有自己的 API 密钥：
1. 点击右上角"设置"按钮
2. 填写 API 密钥
3. 保存即可

**就这么简单！**

## 📖 详细功能说明

### 视频生成
1. 切换到"视频工具"标签
2. 输入视频描述（越详细越好）
3. 选择模型和时长
4. 点击"生成视频"
5. 等待生成完成，可以看到实时进度

**支持的模型：**
- `sora_video2` - 标准模型（推荐）
- `sora_video2-landscape` - 横屏模型
- `sora_video2-portrait` - 竖屏模型

### 图像生成
1. 切换到"图像工具"标签
2. 输入图像描述
3. 选择尺寸和数量
4. 点击"生成图像"

### 角色功能
1. 切换到"角色工具"标签
2. 上传包含角色的视频（⚠️ 不能是真人）
3. 设置角色出现的时间范围（1-3秒）
4. 点击"创建角色"
5. 创建成功后，在"角色视频生成"中使用

**支持的角色模型：**
- `sora-2-landscape-hd` - 横屏高清（默认推荐）
- `sora-2-portrait-hd` - 竖屏高清
- `sora-2-characters` - 角色专用
- 其他 3 种模型

## ⚙️ API 配置说明

### 简单模式（推荐）
只需填写一个 API 密钥，所有功能都能使用：

```
API 密钥: your-api-key-here
```

**留空使用内置免费 API！**

### 高级模式
如果你有独立的 Pro API 密钥，可以点击"显示高级选项"：

```
标准 API Key: your-standard-key
标准 API URL: https://api.maynor1024.live/ (已自动填写)

Pro API Key: your-pro-key (可选)
Pro API URL: https://apipro.maynor1024.live/ (已自动填写)
```

**优势：**
- 🎯 成本优化：普通功能用标准 API，角色功能用 Pro API
- 🔐 独立管理：两个 API 互不影响
- 💡 智能回退：Pro API 未配置时自动使用标准 API
- ✨ **URL 已内置**：无需手动填写 API 地址，开箱即用

## ❓ 常见问题

### Q: 需要配置 API 吗？
A: 不需要！系统已内置免费 API，直接使用即可。

### Q: 如何使用自己的 API？
A: 点击右上角"设置"按钮，填写 API 密钥即可。

### Q: 什么是高级模式？
A: 高级模式允许你为不同功能配置不同的 API Key，实现成本优化。普通用户不需要使用。

### Q: 角色功能为什么需要 Pro API？
A: 只有 Pro API 支持角色创建和角色视频生成功能。如果你只配置了标准 API，系统会自动使用标准 API 尝试角色功能。

### Q: 视频生成失败怎么办？
A: 
1. 检查网络连接
2. 尝试简化描述
3. 更换模型
4. 查看控制台错误信息
